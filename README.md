# Pit Stop Prediction - ML Project

## Overview
This repository contains machine learning experiments and multiple submission files for predicting whether a Formula 1 car will make a pit stop in the next lap.

The project focuses on binary classification using different machine learning approaches and feature engineering techniques.

---

## Dataset Features
Some important features used in training:

- Lap Number
- Tyre Compound
- Tyre Age
- Fuel Load
- Lap Time
- Position
- Weather Conditions
- Cumulative Degradation

---

## Models Used
- XGBoost
- Random Forest
- Logistic Regression
- CatBoost

---

## Project Contents
This repository includes:

- Multiple model submissions
- Data preprocessing notebooks
- Feature engineering experiments
- Model training workflows
- Prediction files
